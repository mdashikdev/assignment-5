Important FAQ

What is the document object model (DOM)

The Document Object Model (DOM) is a programming interface that represents an HTML document as a tree structure. It provides methods to access and manipulate elements, attributes, and text content of the document.

How do you select an element from the DOM?

To select an element from the DOM, i can use JavaScript's getElementById() method by specifying the element's unique ID. Alternatively, i can use the querySelector() method with a CSS selector to find elements based on various criteria.

What is event delegation in the context of the DOM and why is it useful?

Event delegation involves attaching an event listener to a parent element and then using event bubbling to handle events that occur on its child elements. This is useful because it allows you to efficiently handle events for multiple elements without having to attach individual listeners to each one.

How do you manipulate and element's attributes and styles using the DOM?

To manipulate an element's attributes, i can use setAttribute() and getAttribute() methods. To modify its styles, i can access the element's style property and set individual CSS properties directly or use the classList property to add or remove CSS classes.
